  • git clone path-to-repository
  • git status - статус тeкущего локального репозитория
  • git add path-to-file - Добавить новый файл под контроль гита
  • git commit -am "added new file index.html" зафиксировать изменения и дать им комментарий
  • git push - отправить все законченные изменения в реапозиторий на сервер
  • git pull - забрать все свежие изменения с сервера
  • $ git commit -am "added push-pull instructions to HTML" обновить изменения на внешнем ПК
